Model Details

Deconstruction of the Model:

Data Collection & Cleaning

  • Detecting extraneous information and dropping empty rows
  • Handling Missing and invalid Data Values
  • Useful data sorted then used for model training & features
  • Proper Data Alignment & Scaling measures

Linear Regression

  • Models Linear Relationship between input features & a continous target variable
  • Serves as reference benchmark for initial performance
  • Simple, fast, interpretable & understanding attribute influence
  • Used to validate trends & assess feature importance

Random Forest Regressor

  • For real-time prediction.
  • Fast and reliable for tabular data
  • Able to handle Non-Linear Data
  • Properly handles Missing Data and Outliers

XGBoost Regressor

  • High Performance base learner in stacked models
  • Excellent accuracy and efficiency
  • Built-in regularization to prevent overfitting
  • Handles complex interactions between features

Artifical Neural Network(ANN)

  • Implemented as MLPRegressor(Multi-Layer Perceptron)
  • Meta-learner in stacking
  • Captures Complex, Non-Linear feature interactions
  • Final stage of stacking regressor Pipeline

Stacking Regressor(Ensemble Learning)

  • Combines multiple base models into a stronger unified model
  • Leverages strenghts of different algorithms
  • Improves accuracy and generalization
  • Combines Random Forest & XGBoost for ANN
